Covid Daily is an app that allows people to track COVID-19 cases in an easy-to-use interface to increase awareness. Unfortunately, Play Store has rejected Covid Daily as the App Store Review Guidelines require that medical apps are published by medical institutions, so we've open sourced Covid Daily with the intent that people can use it.

Install Instructions:

  1. Click on the DOWNLOAD button to download this app "COVIDDaily.apk".
  2. Open "CovidDaily.apk" file and Install it on your device.
  3. If installation is restricted by your device, go to settings and enable Install Unknown Apps (or Install Other Apps)
  4. This app uses internet to get updates about COVID-19 cases.
  5. Share with others and spread awareness.

Features:

  • Shows data for the nation along with it's states and union territories. Also includes data in district wise manner.
  • Has a user friendly interface.
  • Follows symptom and precaution details as prescribed by WHO.
  • Pictorial representation of symptoms and precautions.
  • Allows to share data for each state and union territory.
